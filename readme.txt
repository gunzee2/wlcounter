勝敗数をカウントする単機能なツールです。
勝敗数と勝敗履歴をボタンを押すたびテキストファイルに出力するのが特徴です。
OBS(Open Broadcaster Software)などで、勝敗を画面上に表示する用途を想定してます。

使い方：
OBSのテキストソースのプロパティで、「ファイルからテキストを使用(UTF-8」に、
任意で下記のテキストファイルパスを指定してください。

1.counter.txt … 勝敗数(3勝2敗など)
2.history.txt … 勝敗履歴(○☓○など)

次にwlcounter.exeを起動します。
あとはツール上の勝・負ボタンを押すだけでOBSの字幕上に反映されます。
押し間違えた時などはUndoボタンを押すと取り消せます。

ツールの勝敗数表示(黒背景の所)を右クリックすると、リセットできます。
アプリを閉じても勝敗数は保存されるので、負けた時などに押すと良いです。

counter.txtやhistory.txtが無いとエラー吐いて落ちると思います。
あとテキストファイルを書き換えても勝敗データは別の場所(localStorage)に
保存しているので意味がありません。
